🔧 Configuration Guide

Setup instructions for advanced features (Edit your .env file)

To enable premium features, add or update these variables in your .env file in the project root:

OPENAI_API_KEY {% if config.openai_enabled %}✓ Configured{% else %}Not Set{% endif %}

Your OpenAI API key for AI-powered task extraction and priority suggestions

📌 Get your key from OpenAI Platform →
GMAIL_ADDRESS & GMAIL_PASSWORD {% if config.email_enabled %}✓ Configured{% else %}Not Set{% endif %}

Gmail credentials for sending task reminder emails

⚠️ Use an App Password, not your regular Gmail password for security
NOTIFICATION_EMAIL {% if config.recipient_email %}{{ config.recipient_email }}{% else %}Not Set{% endif %}

Email address where you want to receive task reminders and notifications

📧 Receives automated emails for stale tasks and pending notifications
SCHEDULER_ENABLED {% if config.scheduler_enabled %}true{% else %}false{% endif %}

Enable or disable the background task scheduler (true/false)

⏰ When enabled, runs task checks every 30 minutes automatically
📝 How to Edit:
  1. Open your .env file in the project root directory
  2. Add or update the environment variables above
  3. Save the file
  4. Restart the Flask application
  5. Changes will appear here in the System Status section
